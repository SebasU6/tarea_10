position: Permite controlar el método de posicionamiento de un elemento

static: Es el posicionamiento por defecto.

relative: El elemento se posiciona de manera normal, pero luego se desplaza

fixed: El elemento se posiciona con respecto al viewport

absolute: El elemento se posiciona con respecto a su primer ancestro posicionado mas cercano

sticky: El elemento se posiciona de forma similar a relative hasta alcanzar una cierta posición de desplazamiento
