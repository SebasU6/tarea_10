Al usar RGB en CSS, se representan las fuentes de luz, Rojo, Verde, Azul.

Con este valor de color solo se usan valores numericos, por ejemplo RGB(255, 0, 0) muestra el color rojo.

Al usar HEX, se refiere a usar valor hexadecimal.

Con este valor se empieza utilizando el numeral(#) primero re representa el rojo(rr), seguido del verde (gg) y finalmente el azul(bb).

Con este metodo se combinan letras y numeros, en este ejemplo el color del texto es: #65ee81.
